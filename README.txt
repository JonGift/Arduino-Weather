Arduino Weather installation
Please note that you'll need six LED's, seven wires, six resistors, an Arduino and a breadboard.

1. Install Python 3, PySerial, and the Arduino drivers.
2. Open the .uno file. Change the LED ports to the ones you are using.
3. Find the COM port your Arduino's using.
4. Open the Python code and change the COM port.
5. Upload the Arduino code and run the Python script.
